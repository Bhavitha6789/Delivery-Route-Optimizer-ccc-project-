How DP Works Here

01
State Definition

State = (currentNode, visitedMask). The bitmask tracks which nodes have been visited.

02
Overlapping Subproblems

The same (node, mask) pair can be reached via multiple paths — we compute it only once.

03
Optimal Substructure

minCost(u→v) = min over neighbors w of (cost(u,w) + minCost(w→v))

04
Memoization

Results stored in a Map. On revisit, the cached value is returned instantly — O(1) lookup.
